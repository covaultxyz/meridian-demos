Jon Cloud Based Solution

INTERNAL 2026-06-08 alex

Summary

Jon and a developer discussed building a cloud-based SaaS product for a PR firm to scan documents for defamatory statements, starting with a basic tier and scaling later.

Decisions Made

  • Build a cloud-based SaaS product instead of a local executable.
  • Start with a basic tier scanning for defamatory statements (tier one).
  • Use a dedicated server to avoid crashing with scale.
  • Compensate developer at $150 per hour for work.

Commitments

  • Developer will research hosting solutions compatible with Jon's HostGator setup.
  • Developer will report back after research before moving forward.
  • Jon will purchase a domain name for the tool.

Key Discoveries

  • The PR firm processes about 1,000 articles per week in batches of 30-50 Word docs.
  • Uploading to a cloud solution may take a couple minutes for 50 documents, which is acceptable.
  • Notion was recommended as a personal knowledge management tool to track project progress.
