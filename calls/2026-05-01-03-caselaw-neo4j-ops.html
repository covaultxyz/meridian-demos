Phase 2 — Golden Query Hardening

Current p95 latency: 5-25s for full-graph queries. Target: <5s.

Test current latency

```bash

Time a BM25 search

time cypher-shell -u neo4j -p "$CASELAW_NEO4J_PASSWORD" "

CALL db.index.fulltext.queryNodes('case_fulltext', 'arbitration employment')

YIELD node, score

RETURN node.case_id AS id, node.citation_str AS citation, score

ORDER BY score DESC LIMIT 10;

"

Time a precedent chain query

time cypher-shell -u neo4j -p "$CASELAW_NEO4J_PASSWORD" "

MATCH (c:Case {citation_str: 'Circuit City Stores, Inc. v. Adams, 532 U.S. 105 (2001)'})

MATCH (c)-[:CITES]->(cited:Case)

RETURN cited.citation_str, cited.pagerank

ORDER BY cited.pagerank DESC LIMIT 20;

"

```

Hardened query patterns (USE THESE in lex_mcp_server.py)

Pattern 1 — BM25 search with LIMIT before expansion

```cypher

CALL db.index.fulltext.queryNodes('case_fulltext', $query)

YIELD node AS c, score

WHERE score > 0.5

WITH c, score

ORDER BY score * coalesce(c.pagerank, 0.0) DESC

LIMIT $limit

RETURN c.case_id, c.citation_str, c.court_id, c.year, c.pagerank, score

```

Pattern 2 — Jurisdiction-filtered BM25 (MUCH faster than post-filter)

```cypher

CALL db.index.fulltext.queryNodes('case_fulltext', $query)

YIELD node AS c, score

WHERE score > 0.5 AND c.court_id STARTS WITH $circuit_prefix

WITH c, score

ORDER BY score * coalesce(c.pagerank, 0.0) DESC

LIMIT $limit

RETURN c.case_id, c.citation_str, c.court_id, c.year, c.pagerank, score

```

Pattern 3 — Precedent chain (depth-limited, no cartesian product)

```cypher

MATCH (c:Case {case_id: $case_id})

CALL {

WITH c

MATCH (c)-[:CITES]->(cited:Case)

RETURN cited, 'forward' AS direction

UNION ALL

MATCH (c)<-[:CITES]-(citing:Case)

RETURN citing AS cited, 'backward' AS direction

}

WITH cited, direction

ORDER BY coalesce(cited.pagerank, 0.0) DESC

LIMIT $limit

RETURN cited.case_id, cited.citation_str, direction, cited.pagerank

```

Pattern 4 — Judge profile (use DECIDED_BY index)

```cypher

MATCH (j:Judge)

WHERE j.name_first + ' ' + j.name_last CONTAINS $judge_name

OR j.name_last CONTAINS $judge_name

WITH j LIMIT 3

MATCH (j)<-[:DECIDED_BY]-(c:Case)

WITH j, c

ORDER BY coalesce(c.pagerank, 0.0) DESC

LIMIT $limit

RETURN j.name_first, j.name_last, c.case_id, c.citation_str, c.pagerank

```

Anti-patterns (DO NOT USE — cause full graph scans)

```cypher

-- BAD: count THEN filter (scans all 11M nodes first)

MATCH (c:Case) WHERE c.court_id = 'ca9' RETURN count(c)

-- GOOD: use index

MATCH (c:Case) WHERE c.court_id STARTS WITH 'ca9'

RETURN count(c)

-- BAD: no LIMIT on BM25 results before join

CALL db.index.fulltext.queryNodes('case_fulltext', 'arbitration')

YIELD node, score

MATCH (node)-[:CITES]->(cited) -- cartesian product explosion!

RETURN ...

-- GOOD: LIMIT before join

CALL db.index.fulltext.queryNodes('case_fulltext', 'arbitration')

YIELD node, score

WITH node LIMIT 20

MATCH (node)-[:CITES]->(cited)

RETURN ...

```

Phase 3 — Embedding Population

Case nodes don't have embeddings yet. This enables vector similarity search.

```bash

Verify enrichment script exists

ls /opt/meridian/current/scripts/run-caselaw-enrichment.py

Run WITHOUT --skip-embeddings flag (default includes embeddings)

source /etc/lex/env

cd /opt/meridian/current

python scripts/run-caselaw-enrichment.py \

--neo4j-uri bolt://localhost:7687 \

--neo4j-password "$CASELAW_NEO4J_PASSWORD" \

--ollama-endpoint http://localhost:11434 \

--model all-minilm-l6-v2 \

--batch-size 500

Monitor progress (in a separate terminal)

watch -n 30 'cypher-shell -u neo4j -p "$CASELAW_NEO4J_PASSWORD" \

"MATCH (c:Case) WHERE c.embedding IS NOT NULL RETURN count(c)"'

```

**Estimated time**: ~1-2 hours for full 11M node corpus on GTX 1070 Ti.

This runs non-destructively in the background. Vector index (`case_embedding`) already exists —

embeddings will be queryable as each batch commits.

To check embedding health after completion:

```bash

cypher-shell -u neo4j -p "$CASELAW_NEO4J_PASSWORD" "

MATCH (c:Case) WHERE c.embedding IS NOT NULL

WITH count(c) AS embedded

MATCH (c2:Case)

WITH embedded, count(c2) AS total

RETURN embedded, total, round(toFloat(embedded)/total * 100, 2) AS pct_complete;

"

```

Phase 4 — Start / Stop Procedure

**Always stop Delphi and Martin BEFORE stopping Neo4j.** They hold open connections.

```bash

Safe stop

systemctl stop meridian-daemon@delphi meridian-daemon@martin

sleep 5

systemctl stop neo4j

Safe start

systemctl start neo4j

sleep 20 # Neo4j takes 15-20s to fully come up on 11M nodes

systemctl start meridian-daemon@las

sleep 3

systemctl start meridian-daemon@delphi meridian-daemon@martin

Verify Neo4j came up before restarting daemons

cypher-shell -u neo4j -p "$CASELAW_NEO4J_PASSWORD" "RETURN 1;"

```

Phase 5 — Memory Tuning

Big-white has 64GB RAM. Caselaw Neo4j config at `/etc/neo4j/neo4j.conf`:

```

Current recommended settings for 11M node caselaw graph

server.memory.heap.initial_size=16g

server.memory.heap.max_size=16g

server.memory.pagecache.size=24g

```

Leaving ~24GB for OS, Ollama (8-10GB for GTX 1070 Ti VRAM via CUDA), daemons, Docker.

Monitor GC pressure:

```bash

journalctl -u neo4j | grep -i "gc\|heap\|OutOfMemory"

```

If GC stalls appear (>500ms GC pauses), increase heap to 20g and pagecache to 20g.

Do NOT exceed 48g combined (leaves insufficient headroom for Ollama + OS).

Phase 6 — Dump & Restore (Disaster Recovery)

Community Edition requires OFFLINE dump. Daemons must stop first.

```bash

=== DUMP ===

systemctl stop meridian-daemon@delphi meridian-daemon@martin

systemctl stop neo4j

DUMP_DIR=/tmp/caselaw-dump-$(date +%Y%m%d-%H%M)

mkdir -p $DUMP_DIR

neo4j-admin database dump neo4j --to-path=$DUMP_DIR/

Transfer to gateway8 (long operation — use tmux)

borg create root@100.73.1.114:/srv/archive-2/lex/ \

--name "caselaw-dump-$(date +%Y%m%d-%H%M)" \

$DUMP_DIR

systemctl start neo4j

sleep 20

systemctl start meridian-daemon@las meridian-daemon@delphi meridian-daemon@martin

=== RESTORE (disaster recovery only) ===

systemctl stop 'meridian-daemon@*'

systemctl stop neo4j

neo4j-admin database load neo4j \

--from-path=/tmp/restore/caselaw/ \

--overwrite-destination

systemctl start neo4j

sleep 20

systemctl start meridian-daemon@las

sleep 3

systemctl start 'meridian-daemon@*'

Verify restore

cypher-shell -u neo4j -p "$CASELAW_NEO4J_PASSWORD" "MATCH (c:Case) RETURN count(c);"

```
