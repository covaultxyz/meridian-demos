Caselaw Research Capabilities (via Lex MCP Tools)

Station 1A — Case Search

```

caselaw_search(query, jurisdiction=None, court_level=None, limit=10)

```

• BM25 full-text over 11M+ case opinions

• PageRank-ranked (most-cited cases surface first)

• Jurisdiction-aware filtering (1st–11th Circuit, D.C., Federal, Supreme Court)

• Returns: case_id, citation_str, title, court, year, pagerank, summary snippet

Station 1B — Precedent Chain

```

case_precedent_chain(case_id, direction="backward", depth=2)

```

• `backward`: what authorities did this case rely on? (foundational chain)

• `forward`: what cases later cited this? (precedential spread, influence score)

• Depth 1-3 (depth >3 is slow on full corpus without index optimization)

• Flags cases that were later overruled or distinguished

Station 1C — Contradiction Detection

```

caselaw_contradictions(issue_query, jurisdiction=None)

```

• Circuit split identification: same legal issue, opposite holdings

• Overruling chain detection (A overruled B)

• Regulatory/statutory change impact on prior holdings

• Returns: conflict pairs with confidence score
